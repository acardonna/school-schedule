## School Schedule,
This is a program based on the implementation of a genetic algorithm that generates optimized weekly schedules for student groups, intelligently avoiding scheduling conflicts and satisfying complex constraints. The timetables satisfy multiple constraints, including:

- No teacher/room/student conflicts,
- No gaps in schedules,
- Proper room-subject compatibility,
- Correct weekly lesson frequencies per subject,

## Project Overview,
This project used:

- Java 21 - Programming language,
- Maven - Build and dependency management,
- MySQL - Database for storing the program's data,
- JDBC - Java Database Connectivity for database access,
- Log4j2 - Logging framework,
- JUnit 5 - Unit testing,
- Jackson - JSON processing library,
- MVC Pattern - Architectural pattern with clear separation of concerns,
- Service Layer Pattern - Services used for the business logic,
- DAO Pattern - Data Access Objects for database operations
